# E-Commerce Order Engine

## Project Overview
CLI-based backend simulation of an e-commerce system.

## Features
- Product management
- Cart system
- Order processing
- Payment simulation
- Concurrency handling
- Coupon system
- Fraud detection
- Event-driven architecture

## Design Approach
- Layered architecture (model, service, util)
- Locking for concurrency
- Event queue for async processing

## Assumptions
- Single machine simulation
- Random payment success/failure

## How to Run
- Run Main.java
- Use CLI menu
